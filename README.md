✈️ Validação de Candidato a Piloto

Aplicação web simples para validar candidatos com base em critérios definidos.

🚀 Funcionalidade

O sistema analisa:

Idade (25 a 45 anos)
Horas de voo (mínimo 1000)
Nota psicológica (mínimo 8)

Retornando aprovado ou inapto, com os motivos.

🛠️ Tecnologias
HTML5
CSS3
JavaScript
